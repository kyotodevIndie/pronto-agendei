### Installation

1. Clone the repo

```sh
git clone https://gitlab.com/sistemascal/prontoagendei.git
```

3. Install the packages using NPM or Yarn

```sh
yarn
```

or

```sh
npm install
```

4. run the project

```sh
yarn dev
```

or

```sh
npm run dev
```

5. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

6. Happy coding!
